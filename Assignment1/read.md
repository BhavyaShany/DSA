Algorithmic Efficiency & Recursion Toolkit (AERT)

Data Structures – Unit 1 Assignment | ETCCDS202

This project implements core recursive algorithms and data structures in beginner-level Python.
What's included:

aert_toolkit.py – Main Python file (run with python aert_toolkit.py)
report.docx – Complexity analysis and paradigm explanations
output.txt – Console output for all test cases

Topics covered: Stack ADT, Recursive Factorial, Naive & Memoized Fibonacci, Tower of Hanoi, Recursive Binary Search, Big-O analysis, and algorithm paradigms (Divide & Conquer, Dynamic Programming, Greedy).
